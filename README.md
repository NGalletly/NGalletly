### Projects

Borough Books
This is a social book-lending app born from a love of community libraries. I really dove into the deep end with the backend here, engineering a Node.js and PostgreSQL server from scratch to handle everything from user transactions to complex book data. One of the most satisfying parts was integrating barcode scanning for ISBNs and setting up Supabase cloud functions to keep the lending process seamless and real-time.

Emerald’s News
For this one, I built a full-stack news aggregation platform where users can actually engage with the content through voting and comments. My main focus was architecting a reliable API following MVC patterns. I spent a lot of time on the filtering logic and used Test Driven Development (TDD) with Jest to make sure that no matter how much data we threw at the server, the endpoints stayed 100% reliable.
